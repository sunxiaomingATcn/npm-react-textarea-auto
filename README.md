

## React Textarea
## React autosize textarea 🎉🎉🎉

高度自动撑开文本域

#### Install
```javascript
npm i react-textarea-auto
```

#### 代码
```javascript
import AutoTextArea from 'react-textarea-auto'
<AutoTextArea maxLength={300} />

/**
 * params maxLength: 默认3000
 * params placeholder: 默认'请输入'
 * params value: 初始值
 * params onChange
*/
```
